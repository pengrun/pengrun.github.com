引言

搭建Hexo折腾了两天时间，终于算是有个雏形了。

Markdown

虽然才开始结束Markdown，感觉还挺好用的。语法简单明了，而且Mac上得Mou界面相当的简介，而且左右分屏，显示代码所对应出来的效果。

Hexo

Hexo里面涉及的语法也很简单，比如

$ hexo deploy

可以简写成

$ hexo d

一次类推比如：

$ hexo new == hexo n
$ hexo server == hexo s
$ hexo generate == hexo g
